Highlights

Every Platform

WebAssembly core. One binary runs in Node.js, .NET, PHP, Python, browsers, and edge functions — no native extensions, no platform-specific builds.

Your Data Stays Yours

The engine runs in-process, inside your infrastructure. No HTTP calls to an external service — documents are generated on the machine that hosts your application.

Declarative Document Model

Define documents in code, XML, or YAML. Flow layout handles structured content automatically; canvas gives exact coordinate control. Both coexist in the same document.

Data Binding

data-value, data-source, and data-if are first-class element attributes. Bind a JSON or XML file to a template — no external templating engine required.

Design Tokens

Spacing, typography, color, and borders are token-addressable at a single point. Swap the token file to restyle the entire document — white-labeling, themes, brand kits.

VS Code Extension

Schema validation, live PDF preview on save, and code generation — built around the same XSD the engine validates against. Install from Marketplace.
